# Editkaro Site

Professional editing services website built with Node.js and Express.

## Features

- Home page
- Portfolio showcase
- About section
- Contact form
- Responsive design

## Installation

1. Clone or download the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file based on `.env.example`:
   ```bash
   cp .env.example .env
   ```
4. Start the server:
   ```bash
   npm start
   ```

## Development

For development with auto-reload:
```bash
npm run dev
```

## Project Structure

- `public/` - Static files (CSS, JS, images)
- `views/` - EJS templates
- `data/` - Data files (contacts, subscribers)
- `server.js` - Express server configuration
